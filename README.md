->Online Food Ordering System using Java
This project is a mini Java application developed using Object-Oriented Programming (OOP) concepts. The system simulates an online food delivery pricing platform with different order types such as Normal, Express, and Premium orders.

->Features:
- Different order categories
- Delivery charge calculation
- Express delivery rush fee
- Premium membership discount
- Runtime polymorphism
- Method overriding using inheritance

->Technologies Used:
- Java
- Object-Oriented Programming (OOP)

->OOP Concepts Implemented:
- Inheritance
- Polymorphism
- Method Overriding
- Arrays
- Classes and Objects
  
->Order Types:
*Normal Order:
- Standard delivery pricing

*Express Order:
- Additional rush fee added

*Premium Order:
- 10% discount
- No delivery charges

->Sample Output:
- Calculates final bill amount based on:
  - Food price
  - Distance
  - Delivery type
